# Portfolio Site Template

A clean, customizable portfolio site template. Fork or clone this repo and follow the steps below to make it your own.

## Getting Started

Use `Template/template-site/` as your starting point.

## Customization Guide

### `Template/template-site/index.html`
- Replace `Your Name` and update the Home section bio
- Update project titles, descriptions, and modal content
- Add your social links (LinkedIn, GitHub, etc.)

### `Template/template-site/js/site.js`
- Set `TEMPLATE_CONTACT_EMAIL` to your email address for the contact form

### `Template/template-site/files/Resume_Template.pdf`
- Replace the placeholder PDF with your actual resume

### Images
- Add your project screenshots to `Template/template-site/img/`
- Update project image references in `index.html` or swap in your own Unsplash URLs

## Structure

```
Template/template-site/

├── js/
│   └── site.js
├── img/
```
